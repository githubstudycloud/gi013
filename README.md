# gi013 项目

## 安装说明

### 环境要求
- Git
- Node.js (推荐 v18+)
- npm 或 yarn

### 克隆仓库
```bash
git clone https://github.com/githubstudycloud/gi013.git
cd gi013
```

### 安装依赖
```bash
npm install
```

### 运行项目
```bash
npm start
```

### 构建项目
```bash
npm run build
```
